# T-SAFV-API

Backend Express para asociaciones de transporte, miembros, unidades, invitaciones, fiscalizacion, pagos y exportaciones.

## Lectura recomendada

- Contexto funcional y tecnico: CONTEXTO_PROYECTO.md
- Arranque operativo: ARRANQUE_RAPIDO.md
- Matriz app/backend: MATRIZ_APP_BACKEND.md
- Historial de cambios: CHANGELOG.md

## Stack

- Node.js
- Express
- PostgreSQL
- JWT
- Jest + Supertest

## Comandos rapidos

```bash
npm install
npm run dev
npm run test
npm run migrate
```
